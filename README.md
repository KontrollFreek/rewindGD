# rewindGD
A quick script to revert Geometry Dash to version 2.2074

## How do I run this?
Go to the [latest release](https://github.com/KontrollFreek/rewindGD/releases), download rewindGD.exe, and run the executable.

If you cannot run the executable (on Linux, etc.)
1. Ensure [Python 3.x](https://www.python.org/downloads/) is installed
2. [Download](https://github.com/KontrollFreek/rewindGD/archive/refs/heads/main.zip) the source code
3. Extract rewindGD-main.zip into a seperate folder
4. Open a terminal or command prompt into the folder you extracted it to
5. Run `python -m pip install -r requirements.txt`
6. Run `python ./rewindGD.py`
